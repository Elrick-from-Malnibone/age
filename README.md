# age

Утилита для расчёта возраста по дате рождения. Работает с локальным временем и поддерживает разные форматы ввода.

## Установка и запуск

```bash
pip install -r requirements.txt
python main.py
```

## Использование

```bash
python main.py --birthdate 1990-05-15
```

Или интерактивно:

```bash
python main.py
# Введите дату рождения: 15.05.1990
```

Форматы дат: YYYY-MM-DD, DD.MM.YYYY, MM/DD/YYYY.
